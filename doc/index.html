SwiftSearch replaces your default WordPress search with an instant, typo-tolerant search engine powered by Typesense. It's designed for speed, converting your visitors into customers with lightning-fast results.

1. Installation

1

Install Typesense

You need a running Typesense server. You can self-host it using Docker (free) or use Typesense Cloud.

2

Install the Plugin

Upload the swift-search-typesense.zip file to your WordPress Plugins dashboard and activate it.

3

Setup Wizard

Go to the SwiftSearch menu in your admin dashboard. Enter your Typesense credentials (Node URL, API Key) to connect.

2. Configuration & Indexing

Once connected, SwiftSearch provides a simple wizard to manage your index.

API Keys Best Practice

For optimal security, use two separate API keys:

  • Admin Key: Provide this in the plugin settings. This allows WordPress to create schemas and index content.
  • Search-Only Key: Generate this in Typesense and use it for your Frontend configuration if you are customizing the JS manually.

Indexing

Click "Index All" to start the process. Indexing runs in the background using your server's cron system. You can close the tab while it runs.

3. Features

⚡ Instant Search

Search results appear as you type. No page reloads.

🔍 Typo Tolerance

Finds "iphone" even if the user types "iphoen". Powered by Typesense.

📦 WooCommerce Ready

Indexes Products, SKUs, Prices, and Stock Status automatically.

✨ Smart Ranking PRO

Advanced ranking rules that prioritize matches in Titles over Content, and boost In-Stock items.

📚 Synonyms PRO

Define synonyms (e.g., "Parka" = "Jacket") to ensure users always find what they need.

4. Displaying the Search

To display the clean search bar anywhere on your site, use the shortcode:

[swift_search]

This will render the search input. Results will appear in an overlay/dropdown instantly.

5. Developer Hooks (Actions)

You can customize the indexing process using standard WordPress actions.

Async Indexing

The plugin uses swift_search_async_index_post to process posts in the background. You can trigger a re-index programmatically:

// Re-index post ID 123
do_action('swift_search_async_index_post', 123);

5. Activating Pro License

If you have purchased a Pro license, you need to activate it to unlock Smart Ranking and Synonyms.

1

Enter License Key

After installing the plugin, you will see an opt-in screen from Freemius. Enter your License Key there.

2

Access Account

You can manage your license, invoices, and subscription from the Account tab in the plugin settings.

6. Theming & Customization

The search interface is designed to inherit your theme's fonts, but you can override styles using standard CSS.

Common Selectors

/* Main Container */
.swift-search-container { }

/* The Search Input */
.swift-search-input { }

/* The Dropdown/Results Area */
.swift-search-results { }

/* Individual Result Item */
.ss-hit-item { }

/* Highlighted Text (Matches) */
.ss-highlight { 
    background-color: #fef08a; 
    color: black;
}

7. Troubleshooting & FAQ

Result "Connection Failed"

  • Ensure your Typesense server is running and accessible (check Firewall).
  • Ensure your API Key has correct permissions.
  • Check your browser console (F12) for CORS errors.

Schema Mismatch Error

If you see a "Critical Schema Mismatch" error, it means the plugin's data structure has changed (likely due to a Pro upgrade). Simply click Index All to reset the collection.
